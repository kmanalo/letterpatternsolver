About

LetterPatternSolver is a fast, browser-based tool to help you find valid 5-letter words based on letter patterns. It’s designed for Wordle-style puzzles and general letter-pattern searching.

How the solver works

  • Known letters: Enter letters you already know in the 5 boxes (leave unknown positions blank).
  • Include letters: Add letters that must appear somewhere in the word.
  • Exclude letters: Add letters that must not appear in the word.

Results are computed instantly in your browser. There is no account system and no server-side processing for searches.

Word lists and browsing

The solver uses two curated word lists: a set of common answer-style words and an expanded set of valid guesses. You can also browse static word list pages (for example, words starting with a letter or ending in a common suffix) from the Word lists page.
